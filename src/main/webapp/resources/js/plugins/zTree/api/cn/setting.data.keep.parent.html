Booleansetting.data.keep.parent

概述[ 依赖 jquery.ztree.core 核心 js ]

zTree 的节点父节点属性锁，是否始终保持 isParent = true

默认值：false

Boolean 格式说明

true / false 分别表示 锁定 / 不锁定 父节点属性

如果设置为 true，则所有 isParent = true 的节点，即使该节点的子节点被全部删除或移走，依旧保持父节点状态。

setting 举例

1. 需要锁定父节点状态

var setting = {
	data: {
		keep: {
			parent: true
		}
	}
};
